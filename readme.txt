== Description ==

Boilerplate for providing theme support for Auto Load Next Post via a plugin.

**Requires**

* Auto Load Next Post v1.5.0 or above.

== Installation ==

Installing "Auto Load Next Post: Support Boilerplate" can be done either by searching for "Auto Load Next Post: Support Boilerplate" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress.
